Краткое описание

В рамках данного курсового проекта была разработана информационная система для салона красоты. В состав разработанной системы входят frontend, backend и база данных. Система размещена на хостинге факультета информационных технологий.

Целью данного курсового проекта является расширение информационного поля небольшого салона красоты, а именно формирование информационной системы для более удобного способа информирования клиентов об оказываемых услугах и способа ведения учёта заказов.

Стек используемых технологий

Веб-приложение: Vue.js

Серверное приложение: Express.js

База данных: mySQL
